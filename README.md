# PDF Cutter

Цей інструмент дозволяє інтерактивно вибирати розділи з PDF-файлу та зберігати їх як окремі PDF-документи. Він використовує зручний термінальний інтерфейс (TUI) для навігації по змісту книги.

## Особливості

- **Інтерактивне дерево змісту**: Відображає структуру PDF-файлу (розділи, підрозділи).
- **Гнучкий вибір**:
    - Якщо вибрати **згорнутий** розділ, він буде збережений як один файл (разом із усіма підрозділами).
    - Якщо вибрати **розгорнутий** розділ, можна вибирати окремі підрозділи для збереження.
- **Автоматичне іменування**: Файли називаються відповідно до назв розділів.

## Встановлення

1. **Створіть віртуальне середовище** (рекомендовано):
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   ```

2. **Встановіть залежності**:
   ```powershell
   pip install -r requirements.txt
   ```

## Використання

Запустіть скрипт, передавши шлях до PDF-файлу:

```powershell
python main.py "шлях\до\вашого\файлу.pdf"
```

Наприклад:
```powershell
python main.py .\unix.pdf
```

### Керування в інтерфейсі

- **Стрілки ВГОРУ / ВНИЗ**: Навігація по списку.
- **Стрілка ВПРАВО**: Розгорнути гілку (показати підрозділи).
- **Стрілка ВЛІВО**: Згорнути гілку.
- **ПРОБІЛ**: Позначити/зняти позначку з розділу (зелений колір = вибрано).
- **ENTER**: Підтвердити вибір та почати розділення файлу.

## Як це працює

1. Скрипт зчитує структуру (Outline) PDF-файлу за допомогою бібліотеки `pypdf`.
2. Будується деревоподібна структура, де для кожного розділу обчислюється початкова та кінцева сторінка.
3. Запускається інтерфейс на базі бібліотеки `textual`, який дозволяє користувачу вибрати потрібні частини.
4. Після підтвердження (`Enter`), скрипт проходить по вибраних елементах:
   - Якщо елемент вибраний і згорнутий -> експортується весь діапазон сторінок цього елемента (включно з дітьми).
   - Якщо елемент вибраний і розгорнутий -> скрипт перевіряє його дітей.
5. Вибрані сторінки зберігаються у нову папку з назвою оригінального файлу.
